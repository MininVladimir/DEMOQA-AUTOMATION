Feature: REG_API_01
Owner: Vladimir Minin

Scenario:   [REG][API] Registration user
    When:   Send request to registration service:
            ``
            Request method: POST
            Request URI:    https://demoqa.com/Account/v1/User
            Headers:        Content-Type=application/json
            Body:           {
                                "userName": "{username}",
                                "password": "{password}"
                            }
            ``
    Then:   Get response from registration service:
            ``
            Status:         201 Created
            Content-Type:   application/json;
                            charset=utf-8
            Body:           {
                                "userID": "{userId}",
                                "username": "{username}",
                                "books": []
                            }
            ``
